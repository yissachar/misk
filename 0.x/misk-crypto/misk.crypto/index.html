    S3KeyResolver class S3KeyResolver :ExternalKeyResolver
    S3KeyResolver implements an KeyResolver that fetches Tink keysets from an S3 bucket. Keysets are indexed by an alias and a region, and are encrypted with a key in the KMS using an envelope key encryption scheme. Each Keyset is protected by a KMS key in each service region.
